InsightSelf HTML 原型

以下为静态 / 轻交互示意页面，与 PRD《InsightSelf 简单PRD》及扩展 HTML PRD 对齐。直接在浏览器中打开链接即可。

  • 登录 / 注册 — 校验规则与密码可见性
  • 注册后 · 个人信息登记 — 性别、星座只读、语言 CH/EN
  • 主页 · 更懂自己 — 三类聚合卡片、下拉说明、AI 底部弹层与二次确认（可点击）
  • 生辰八字 — 一句话评价、时辰缺失横幅、五行/十神、折叠建议
  • 星座解读 — 日月升标题、动态看板、分层折叠建议
  • 信息测评列表 — 问卷状态与二级页说明
  • 测评二级页 · 作答 — 单屏题目 + 进度条 + 返回确认（可点击）
  • 测评二级页 · 结果解读 — 维度条 + 折叠解读 + 分享按钮（静态）
  • 个人中心 — 档案字段、退出登录二次确认（可点击）
